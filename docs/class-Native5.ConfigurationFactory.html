makeConfig Wrap the associative configuration array inside a Configuration class

Note

the ftp configuration should be of the following format: mixed[] $configuration {

Type

string "host" domain name or IP address of ftp host
string "port" port where ftp service is running on the host
string "user" ftp username
string "private_key" ftp user ssh private key for login into ftp host
string "public_key" ftp user ssh public key for login into ftp host
string "directory" directory on ftp host (optional) }
